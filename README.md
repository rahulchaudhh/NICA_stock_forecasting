## NICA Stock Price Forecasting

This project predicts the future stock prices of NICA using historical stock data.

# Project Overview

 The main goal of this project is to:

 Analyze historical stock prices of NICA.

 Forecast future prices for the next 15 days.

Visualize both historical and predicted prices in a clear and informative manner.
